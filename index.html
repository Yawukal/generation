WELCOME TO THE JUNIOR WEB DEVELOPER PROGRAM
a generation USA program


This 11-week program is an intensive, fully online program focused on web development and employment. In this course, you will develop all the technical and professional skills necessary to ensure your success as a Junior Web Developer. Junior Web Developers work in a range of companies in the US and are responsible for assisting with the creation of websites and updating existing web applications using scripting languages.
